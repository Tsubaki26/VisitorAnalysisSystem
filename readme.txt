#インストール環境
OS: Windows11
Python:
	version: 3.11
	外部ライブラリ: ”requirements.txt”を参照
MySQL:
	version: 8.1.0
	データベース，テーブルの作成等は”データベース作成.txt”を参照


#アプリ操作方法
1．"visitor_analysis_system.py"を実行するとメインウィンドウが立ち上がる．
2．"画像ファイルを選択"ボタンを押し，画像を選択すると自動的に認識が始まる．
	左上に入力画像，左中央に画像情報（画像サイズ，撮影日），左下にログ，右上に認識結果が表示される．
3．"来場者の分布を表示"ボタンを押すとグラフウィンドウが立ち上がり，指定期間の地域のヒストグラムが表示される．
	年間，月間，日間から選択できる．チェックボックスにより，グラフに鳥取県を含めるかを選択できる．
	ウィンドウ下部の"進む"，"戻る"ボタンを押すと，日付を移動できる．


#プログラム解説
visitor_analysis_system.py:
	・GUIの構築

(recognitionフォルダ)
NPrecognition_v3.py:
	・preprocessing.pyで基本的な前処理を行う．
	・画像をナンバープレートの各要素に分割する．
	・文字認識を行い，地名とその確率，分類番号，平仮名，一連指定番号，各処理時間を返す．
	・images/output_imagesに分割した画像が保存される．

(MyLibraryフォルダ)
preprocessing.py:
	・リサイズ，適応的ヒストグラム平坦化，2値化などを行う．

control_db.py:
	・データベースにナンバープレート情報を格納
	・データベースを参照しグラフを作成

rc_area.py，rc_kana.py，rc_num.py:
	・学習させたモデル（myModelsフォルダ）を用いて文字認識

split_histgram.py:
	・ヒストグラムを用いてナンバープレート画像を各要素に分割

split_hough.py:
	・ハフ変換を用いて数字を1桁ずつ分割しリスト化
